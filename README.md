# 🐌 snail-bet


a terminal-based snail racing game built with python. bet on your favorite snail and see who reaches the finish line first!


##  how to play


### windows (recommended)

if you are on windows, you don't even need python installed. 

1. go to the **[releases](https://github.com/jamallyemin/snail-bet/releases/latest)** page.

2. download `snails.exe`.

3. run it and start betting!
